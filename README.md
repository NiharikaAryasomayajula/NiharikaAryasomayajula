Has a live PWA at [niharikaaryasomayajula.github.io/Bot-Hire](https://niharikaaryasomayajula.github.io/Bot-Hire). Built it during my own job search, which felt appropriate.

Dual-schema PostgreSQL database separating raw EHR ingestion from curated feature tables, Python ETL with validation at each stage, feeding into a PyTorch transformer pipeline.
